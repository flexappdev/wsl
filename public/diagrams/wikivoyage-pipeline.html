DIAGRAM · INGEST PIPELINE

198 countries, two sources, one snapshot.

The wikivoyage ingest walks every country in SEED.allCountries, asks Wikivoyage first, falls back to Wikipedia, generates a FLUX hero, and writes a single public/data/wikivoyage-countries.json snapshot that ships with the build.

The accent path is the production flow; muted nodes are the parallel FLUX pipeline that runs once per ingest. The snapshot ships in the Vercel build — no runtime API calls.
